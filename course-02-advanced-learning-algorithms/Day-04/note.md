## Topics Learned

### Neural Networks Intuition

- Neural networks are inspired by the human brain.
- They learn patterns from data and make predictions.
- They are used in image recognition, speech recognition, and many AI applications.

### Neural Network Models

- Neural networks consist of layers.
- Input Layer → Hidden Layer(s) → Output Layer.
- More hidden layers allow the model to learn more complex patterns.

### Forward Propagation

- Forward propagation is the process of making predictions.
- Data flows from the input layer through hidden layers to the output layer.

### TensorFlow Implementation

- TensorFlow is a popular framework for Machine Learning and Deep Learning.
- It simplifies building and training neural networks.
- Models can be created using layers and trained using data.
