# Day 10 — TensorFlow Implementation

## What I Learned

Today I learned how neural networks are implemented using TensorFlow.

TensorFlow is one of the most popular frameworks for building and training Machine Learning and Deep Learning models.

---

## Inference in Code

Inference means using a trained model to make predictions.

Example:

Input → Neural Network → Prediction

The model takes data as input and produces an output based on what it learned during training.

---

## Data in TensorFlow

Before training a model, data must be prepared and organized.

TensorFlow works with numerical data and tensors.

A tensor is a data structure similar to an array or matrix that stores values used by the model.

---

## Building a Neural Network

A neural network can be created by defining layers.

Basic structure:

Input Layer → Hidden Layer → Output Layer

TensorFlow provides tools that make building neural networks easier.

## Key Takeaways
