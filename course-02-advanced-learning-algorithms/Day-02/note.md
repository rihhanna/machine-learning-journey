# Day 9 — Neural Network Models

## What I Learned

Today I learned how neural networks are structured and how they make predictions.

### Neural Network Layers

A neural network consists of layers of neurons:

Input Layer → Hidden Layer → Output Layer

Each neuron receives information, performs calculations, and passes the result forward.

### More Complex Neural Networks

Neural networks can contain multiple hidden layers.

More layers allow the model to learn more complex patterns from data.

Examples include:

- Image Recognition
- Speech Recognition
- Natural Language Processing
- Recommendation Systems

### Forward Propagation (Inference)

Inference is the process of making predictions using a trained neural network.

Information flows through the network from the input layer to the output layer.

This process is called Forward Propagation.

### Key Takeaways

- Neural networks are built from layers of neurons.
- Hidden layers help learn complex relationships.
- More layers can solve more difficult problems.
- Forward propagation is how predictions are generated.
