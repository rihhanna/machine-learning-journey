## 🧠 Motivation for Classification

Unlike regression, which predicts numerical values, classification predicts categories or classes.

### Examples:

* Spam or Not Spam
* Pass or Fail
* Malignant or Benign Tumor
* Customer Will Churn or Stay

Classification helps answer "which category?" rather than "what number?"

---

## 📊 Logistic Regression

Despite its name, Logistic Regression is used for classification problems.

It predicts the probability that an example belongs to a particular class.

### Example:

* Probability = 0.90 → Positive Class
* Probability = 0.20 → Negative Class

The output is always between 0 and 1.

---

## 🎯 Decision Boundary

A decision boundary is the line (or surface) that separates different classes.

For example:

* Probability ≥ 0.5 → Class 1
* Probability < 0.5 → Class 0

The decision boundary determines how the model classifies new examples.
