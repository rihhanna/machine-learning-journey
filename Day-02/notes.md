# Day 2 — Gradient Descent & Cost Function

## 📌 What I Learned

Today I explored how machine learning models improve their predictions using optimization techniques.

### 🔹 Cost Function

The cost function measures how wrong the model’s predictions are compared to the actual values.
The goal is to minimize this cost to make the model more accurate.

* High cost → poor predictions
* Low cost → better predictions

---

### 🔹 Gradient Descent

Gradient Descent is an algorithm used to find the best values for the model parameters (w and b).

It works by:

* Starting with initial values
* Repeatedly adjusting them step by step
* Moving in the direction that reduces the cost

The objective is to reach the lowest point (minimum cost).

---

### 🔹 Learning Rate (α)

The learning rate controls how big each step is during optimization.

* Small learning rate → slow but stable
* Large learning rate → faster but may overshoot

Choosing the right learning rate is important for efficient learning.

---

## 🧠 Reflection

This topic was challenging at first, especially understanding how the algorithm updates parameters. However, visualizing it as moving down a curve to reach the lowest point helped me understand the concept better.
