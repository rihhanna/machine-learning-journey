# Day 1 - Introduction to Machine Learning

## What is Machine Learning?
Machine Learning is learning from data to make predictions.

## Supervised Learning
- Uses labeled data
- Two types:
  - Regression (predict numbers)
  - Classification (predict categories)

## Linear Regression
- Predicts numbers using a straight line
- Formula: y = wx + b
